Welcome to Mirror

About Mirror

This site is a public repository of free and open source software. It is located at Clarkson University in Potsdam, NY, US. This site strives to provide exposure to open source software and various open source operating systems. Currently, this site mirrors several Linux distributions, some third party repositories, and even the Linux Kernel. You can find out more information about everything we mirror by choosing the category that you are interested in.

Linux Distros

We host around 20 distributions including Debian, Ubuntu, Arch Linux, Linux Mint, and many more. Many are official mirrors and the majority are updated 4 times a day from upstream mirrors. Learn more on the Distributions page.

GNU Software

We also host a large quantity of software maintained by the GNU Project. you can find it on the Software page.

Linux Kernel

Versions of the Linux Kernel (Up to 4.5.X) can also be found on the Software page, found in the same place is documentation and souce for the kernel.

History

The newest iteration was completed in April 2016, but we have had Mirror running since 2002. If you want to learn more about the newest iteration or the history behind the project, Visit the History page.
